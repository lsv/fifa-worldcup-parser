Usage
-----

Just run `$data = (new Lsv\FifaWorldcupParser\Parser)->parse();`, now `$data` will be a `Lsv\FifaWorldcupParser\Model\Data` object

Install
-------

`composer require lsv/fifa-worldcup-parser`
